Design decisions

The wordmark

Playfair Display 900 weight, lowercase. The gold period is the only brand accent — small but distinctive. The day context line above it grounds the screen without taking space.

Hero card

Changes when a category is active — it promotes the top recipe in that filter, not a static featured recipe. Hero becomes a navigation cue, not decoration.

Category tiles

Visual tiles beat text chips. Flag emoji gives instant cultural recognition. The row scrolls — 9 cuisines accessible without any overflow chrome. Active state is rust-tinted, not a filled pill.

Recipe rows vs grid

List rows show chef + cuisine + time in one scan, no tap needed. Grid cards require tapping to get metadata. For a library this curated, list gives faster decisions.

Gold cuisine tag

The small uppercase gold tag above each recipe title is the visual connector between the category tiles and the list. It tells you exactly what tradition the dish belongs to before your eye reaches the title.

Engineer notes

No new tokens needed. All existing v0.7 values. Category tiles replace the current chip row 1:1. Hero card is a new component above the list, conditionally shows planned recipe or top result of active filter.
