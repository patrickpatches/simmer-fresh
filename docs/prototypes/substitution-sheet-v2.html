Accessibility requirements — never colour alone

Every pill carries three signals

1. Colour — green / yellow / red
2. Icon — ✓ / ≈ / ⚠
3. Text label — "Great swap" / "Some difference" / "Noticeable change"

All three must be present on every rendered pill. Removing any one of them for space reasons is not acceptable.

Accessibility labels (React Native)

Each pill gets accessibilityLabel set to the full quality description:

"Great swap — pancetta works as well as guanciale"

Engineer derives this by combining the pill label + the substitution's change_description field. TalkBack reads the full string; sighted users read the pill.

Ingredient row swapped state

When a swap is active, the ingredient row uses border + bg tint in the swap colour plus a text label ("↩ swapped") and a matching icon (✓ / ≈ / ⚠). The colour alone on the row border is not enough — the text cue is mandatory.
